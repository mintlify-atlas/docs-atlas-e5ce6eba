## MedicalInsights

The `MedicalInsights` model represents the complete AI-generated analysis output for a medical case, combining case summary, SOAP notes, diagnosis, and confidence scoring.

### Fields

<ResponseField name="case_summary" type="CaseSummary" required>
  Comprehensive summary of the medical case
  
  <Expandable title="CaseSummary fields">
    <ResponseField name="comprehensive_summary" type="string" required>
      Detailed narrative summary of the entire case
    </ResponseField>
    
    <ResponseField name="key_findings" type="List[string]" required>
      List of the most important clinical findings
    </ResponseField>
    
    <ResponseField name="patient_context" type="PatientData" required>
      Patient demographic information
    </ResponseField>
    
    <ResponseField name="doctor_notes" type="string">
      Doctor's clinical observations
    </ResponseField>
    
    <ResponseField name="lab_summary" type="string">
      Summary of laboratory findings
    </ResponseField>
    
    <ResponseField name="radiology_summary" type="string">
      Summary of radiology findings
    </ResponseField>
    
    <ResponseField name="confidence_score" type="float" required>
      AI confidence score (0.0 to 1.0)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="soap_note" type="SOAPNote" required>
  Structured clinical documentation in SOAP format
  
  <Expandable title="SOAPNote fields">
    <ResponseField name="subjective" type="string" required>
      Patient's reported symptoms and history
    </ResponseField>
    
    <ResponseField name="objective" type="string" required>
      Observable and measurable findings
    </ResponseField>
    
    <ResponseField name="assessment" type="string" required>
      Clinical assessment and interpretation
    </ResponseField>
    
    <ResponseField name="plan" type="string" required>
      Treatment and follow-up plan
    </ResponseField>
    
    <ResponseField name="confidence_score" type="float" required>
      AI confidence score (0.0 to 1.0)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="primary_diagnosis" type="Diagnosis" required>
  Primary diagnosis with supporting evidence
  
  <Expandable title="Diagnosis fields">
    <ResponseField name="primary_diagnosis" type="string" required>
      Name of the primary diagnosis
    </ResponseField>
    
    <ResponseField name="icd_code" type="string">
      ICD-10 or ICD-11 code for the diagnosis
    </ResponseField>
    
    <ResponseField name="description" type="string" required>
      Detailed description of the diagnosis
    </ResponseField>
    
    <ResponseField name="confidence_score" type="float" required>
      AI confidence score (0.0 to 1.0)
    </ResponseField>
    
    <ResponseField name="supporting_evidence" type="List[string]" required>
      List of evidence supporting this diagnosis
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="overall_confidence_score" type="float" required>
  Overall AI confidence score for the entire analysis (0.0 to 1.0)
</ResponseField>

<ResponseField name="generated_at" type="datetime" required>
  Timestamp when the insights were generated (ISO 8601 format)
</ResponseField>

### Confidence Scoring

The model includes multiple levels of confidence scoring:

- **overall_confidence_score**: Overall confidence in the entire analysis (0.0-1.0)
- **case_summary.confidence_score**: Confidence in the case summary
- **soap_note.confidence_score**: Confidence in the SOAP note generation
- **primary_diagnosis.confidence_score**: Confidence in the primary diagnosis

All confidence scores are validated to be between 0.0 and 1.0 using Pydantic's `Field(ge=0.0, le=1.0)` constraint.

### Timestamp Format

The `generated_at` field uses ISO 8601 datetime format and is automatically populated with the current timestamp when the insights are generated.
